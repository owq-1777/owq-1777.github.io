什么是 Web3?

在了解Web3之前，需要先了解下互联网的历史。

  • Web 1.0(1990-2004)
    互联网诞生时，是和杂志一样的属性，静态的，只读的，没有用户交互，这一段时期的互联网被称为"只读网络"
  • Web 2.0(2004-Now)
    随着社交媒体的出现，网络不再是只读的，而且进化成可读可写，交流变成双向的了，但用户创建的内容被保存在运营的平台商上, 物理上并没有拥有它。
  • Web3(Future)
    在 Ethereum 发布不久后，由 Ethereum 创始人 Gavin 提出的，它使用区块链，加密货币，和 NFTs以所有权的形式给予用户权力, 用户真正拥有了在互联网掌握自己东西的权力。

如果 Web 1.0 是用户读取互联网，Web 2.0 是用户写入互联网，那么 Web3 就是用户生活在互联网。

  1. Web1 代表了互联网的早期阶段，是PC互联网时代的代表性产品，如国际上的 雅虎 和 谷歌，以及国内的门户网站，如 搜狐 和 网易。这些网站的主要功能是汇集信息，用户可以在网站上点击浏览这些信息。
  2. 移动互联网的兴起改变了信息生产的方式，不再仅限于门户网站，全球数亿用户成为信息的生产者。用户不仅可以在互联网上获取信息，还可以自己生成信息。代表性的产品包括国外的 Facebook 和 Twitter，以及国内的 微信** 和 抖音，它们属于Web2时代的互联网产品。这些巨头平台吸引了大量用户，通过广告实现商业化闭环，形成了典型的平台型经济模式。在这个模式下，平台通过提供免费服务来吸引用户，并未经用户允许将用户的信息卖给商家以盈利。
  3. 那么，Web3 又是什么呢？Web3的核心在于用户拥有自己在互联网上的 信息产权。从信息产生的瞬间开始，它就归属于用户。Web3实际上是信息所有者的经济体系。在当前阶段，用户无需依赖大型平台注册账户。一旦用户创建了自己的 数字钱包，就可以在Web3世界中自由使用各种应用程序。钱包的概念和使用将在后面解释。理论上来说，您的数字钱包仅属于您自己，不受任何公司或组织的控制。与Web2不同，如果您在某个游戏中被封号或游戏公司倒闭，您的虚拟资产仍然安全存在于您的数字钱包中。这正是Web3的特点，它强调用户的信息所有权，并建立了原生的价值网络。

简单点说， Web3 就是所有权的改变。

Web3 主要特点

  1. 去中心化

    Web3的核心概念之一是去中心化，即消除传统互联网中的中心化权威和控制，将权力下放到网络的多个节点。这有助于减少单点故障，提高网络的抗攻击性，并增强用户的控制权。

  2. 数字主权

    Web3推崇用户拥有自己的数据和身份，通过加密技术实现数字主权。用户可以更好地掌控自己的个人信息，选择与谁分享以及如何分享。这也是Web3的核心

Web3 相关产物

  1. 加密货币

    Web3 使用加密货币在线消费和汇款，不依赖于传统银行或者其他有着类似于银行功能的第三方机构。就如比特币和以太坊, 这也是资本入驻Web3的第一条道路。

  2. 智能合约

    智能合约就是自动执行的合同，无需中介。这些合同可以用于各种用例，如去中心化金融（DeFi）、数字艺术市场和供应链管理，为各种交易提供可信度和透明度, 相当于无需信任（不需要第三方信任背书）。

  3. DApp

    Web3 支持开发和使用去中心化应用程序，这些应用程序不依赖于中心化的服务器，用户拥有数据和控制权。这种方式有助于消除数据垄断和滥用问题。

  4. NFT

    Web3通过非同质化代币（NFT）等技术赋予数字物品和资产真实的所有权，使数字世界中的资产具有独特性和价值。
