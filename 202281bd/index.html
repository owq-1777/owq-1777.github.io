启动Win-Kex

Win-KeX 有三种启动模式

  • 窗口模式
    在专用窗口中启动 Kali Linux 桌面.

  • 远程连接（推荐）
    微软自带的远程桌面工具，用过的都说好~

  • 无缝模式
    在 Windows 和 Kali 应用程序和菜单之间共享 Windows 桌面.

注: 后面参数-s是启动声音支持.
